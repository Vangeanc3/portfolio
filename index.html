Portfólio

Banpará

Resumo:

Durante meu estágio no Banpará, recebi a tarefa em conjunto com outro estagiário, de fazer um sistema web que automatizava a validação de contratos com base em pontos de função. Esses contratos que eram entregues em uma planilha do excel, em um padrão não muito agrádavel, assim, dificultando o desempenho dos análistas

Funcionalidades:

  • Telas de cadastro e login de usuários
  • controle e gestão de usuários, com poder de gerenciar os usuários do sistema(CRUD).
  • Leitura de planilhas, com upload e download de arquivos via array de bits(CRUD)
  • Controle e gestão de fornecedores e sistemas de terceirizados do Banco(CRUD)

Tecnologias usadas:

  • ASP.NET Core
  • Dapper
  • SQL Server
  • Blazor WASM
