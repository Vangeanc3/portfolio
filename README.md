## CSS Units

Unidades de medidas do CSS

Layout Fixo
px - Pixels

Layout Fluido 
% - Porcentagem
auto - Automática
vh - Viewport Height
vw - Viewport Width

Textos fixos
1px = 0.75pt 
4px = 3pt
8px = 6pt
16px = 12pt

Texto fluidos
em - multiplicado pelo pai
rem - multiplicado pelo root
